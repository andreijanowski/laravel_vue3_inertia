## ⚡️ Technical Stack
1. Laravel
2. Vue 3
3. Vite 3
4. Tailwind CSS
5. Inertiajs

## ⚡️ How to install
1. `composer install`
2. `cp .env.example .env`
3. `php artisan key:generate`   
4. `npm install`
5. `php artisan migrate`
6. `php artisan db:seed`
7. `php artisan serve`
8. `npm run dev`
9. `visit http://localhost:8080`
